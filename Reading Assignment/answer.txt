+) When should accessor methods be used?
Answer: It should be used when you want to access the value of an object's private fields from outside the clasa. They provide controlled access to an object's properties, ensuring that the data remains consistent and encapsulated. They are especially useful when you need to read the data of an object but don't want to allow direct modification.

+) If you create a constructor method to build a DVD by title then create a constructor method to build a DVD by category. Does JAVA allow you to do this?
Answer: Yes, Java allows you to have multiple constructors in a class by using constructor overloading method.
